# vue-project
## 1、安装 webpack
```
 npm install webpack webpack-cli -g
 ```
 
 ## 2、安装 vue-cli
 ```
 npm install -g vue-cli
 ```
 
 ## 3、用vue-cli来构建项目
 ```
 vue init webpack projectName
 ```
 
 ## 4、安装依赖
 ```
 cd projectName
 npm install
```
 
 ## 5、启动项目
 ```
 npm run dev
 ```
 
 ## 6、打包上线
 ```
 npm run build
```
